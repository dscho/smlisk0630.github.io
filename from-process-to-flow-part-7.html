From Process to Flow: Part 7

Here we are: the last task necessary to get your Flow ready for Prime Time! It’s time to create your new Global Action.

Create a Global Action

  1. Go to Setup and search for “global.” Click “Global Actions.”
  2. Click the "New Action" button.
  3. For Action Type, choose "Lightning Component."
  4. For “Lightning Component,” find the name of the Component you just created.
  5. For height, enter "50px."
  6. Do not choose a Standard Label Type.
  7. For Label, enter the name you want users to see to access the Action, e.g. “New Case.”
  8. When a Label is entered, the Name field should auto-populate with an API name. Make a note of this.
  9. Enter a Description, then click "Save." Your Action should look like mine in the screenshot above.

Add the Action to the Publisher Layout

  1. Once again, go to Setup and search for “global,” but this time click “Publisher Layouts.”
  2. Click “Edit” beside your layout.
  3. If you have not already removed the other New Case global action from the “Salesforce Mobile and Lightning Experience Actions” layout, remove it now.
  4. At the top, click “Mobile & Lightning Actions.”
  5. Find the Action you created. If it has the same Label as another action, hover over the Actions and find the correct one via the API Name.
  6. Drag the Action into the “Salesforce Mobile and Lightning Experience Actions” layout and click “Save.”

And that’s it. You and your users should now be able to use your new Flow in a Global Action to submit a Case. Be sure to test it by creating Cases!
